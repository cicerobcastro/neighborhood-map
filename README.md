Project created using React JS, Foursquare and Google Maps API. <br/>

## Dependencies
*Google Maps API
*Foursquare API


## How to Run
    
* install node and npm
* clone this repo
* open terminal and go to the directory of project
* run npm install
* run npm start
* open browser to localhost:3000
